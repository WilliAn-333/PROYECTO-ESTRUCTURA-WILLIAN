Asignatura: Estructura de Datos

Basado en el sílabo oficial ESPOCH

Unidad 1 Unidad 2 Unidad 3 Unidad 4

Unidad 1: Fundamentos e Introducción a las Estructuras de Datos

Resultado de aprendizaje: Identificar características y fundamentos de las estructuras de datos mediante el análisis de bibliografía y casos prácticos.

  • Introducción (Historia, Características, Ventajas y Desventajas)
  • Tipos de datos (Primitivos, Compuestos, Abstractos)
  • Datos compuestos (Vectores, Matrices, Registros)
  • Fundamentos de algoritmia (Conceptos, Representación, Ciencias de la Computación)

Unidad 2: Estructuras de Datos Lineales

Resultado de aprendizaje: Analizar y aplicar estructuras lineales (listas, pilas y colas) para resolver problemas computacionales.

  • Pilas (definición, operaciones, aplicaciones)
  • Colas (tipos, operaciones, aplicaciones)
  • Listas (nodos, tipos, representación, operaciones)

Unidad 3: Estructuras de Datos No Lineales

Resultado de aprendizaje: Aplicar estructuras no lineales (árboles y grafos) para solucionar problemas computacionales.

  • Visualización de algoritmos
  • Árboles (definición, recorrido, métodos)
  • Grafos (definición, tipos, propiedades)

Unidad 4: Búsqueda y Ordenamiento

Resultado de aprendizaje: Conocer y aplicar métodos de búsqueda y ordenamiento en distintas estructuras de datos.

  • Análisis de algoritmos (Complejidad temporal y espacial)
  • Ordenamientos: Burbuja, Quicksort, Mergesort
  • Búsquedas: Secuencial, Binaria, Hash
